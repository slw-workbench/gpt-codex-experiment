# Food Promart E-Tax Form

คำร้องขอใบกำกับภาษีอิเล็กทรอนิกส์ตัวอย่างสำหรับ Food Promart

## การติดตั้ง

```bash
npm install
npm run dev
```

## สคริปต์อื่น ๆ

- `npm run build` – สร้างไฟล์สำหรับ production
- `npm run preview` – แสดงตัวอย่าง production
